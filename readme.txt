Criei este programa com o objetivo de resolver o Problema do Caixeiro Viajante (TSP - Travelling Salesman Problem).

Esse problema consiste em determinar a melhor rota entre vários endereços.
Desenvolvi três estratégias para resolver esse problema, sendo a primeira a mais simples, porém menos eficiente.

O programa é baseado no uso de uma planilha .xlsx contendo as coordenadas de latitude e longitude.

Este é o meu primeiro projeto em Python que aborda um problema real. Espero que seja útil para futuros desenvolvedores e também para o meu eu futuro.

Created by Endrio Marcos de Liz Oliveira

I developed this project with the aim of solving the Traveling Salesman Problem (TSP).

This problem involves determining the most efficient route among N addresses.
I have devised three strategies to achieve this goal, with the first one being the simplest but less efficient.

The entire project is based on the use of an .xlsx file containing the latitude and longitude coordinates of the required addresses.

This is my first Python project that solves a real-life problem. I hope it helps future developers and myself in the future.
